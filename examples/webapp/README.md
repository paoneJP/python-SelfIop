Demo RP for Self-Issued OpenID Provider
=======================================

## What is this?

   - use for testing SelfIop.
   - shows how to write RP.


## Requirements

 * Python3.3+
 * following python libraries
   - PyCrypto
   - bottle


## How to play it

Start application.

    $ python3 demoapp.py

Access to following URL.

    http://localhost:8081/
